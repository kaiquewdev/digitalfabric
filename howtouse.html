How to use Digital Fabric QR Code Ganerator:

*General Usage

The images produced will be shown with a limit of 1000px by 1000px but can be saved, with the maximum limit which is 5000px by 5000px.
Use the option in the context of the right to save as an image.
With the dimensions set, you generate a QR code. With each new QR code generated, it takes the values ​​of width and height, so changing values ​​and clicking you can generate QR's with different sizes.

*Qr URL

To generate a QR with the value of a URL, put the value similar to the example below.
Example: http://www.google.com or https://mail.google.com

*Qr Text
